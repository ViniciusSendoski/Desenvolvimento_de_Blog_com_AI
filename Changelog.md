## Version 2.1 
- GitHub page
Settings -> Pages

No branch, selecione main para criar um link para carregar o arquivo Index.html

- Color Hunt

O site https://colorhunt.co/ possui varias palhetas de cores, usado para escolher a palheta de cores atualizada do site. Palheta escolida: https://colorhunt.co/palette/2e073f7a1cacad49e1ebd3f8

- Google Fonts

O site https://fonts.google.com/ possui varias fontes para caracteres do site, a escolhida para o Blog foi: https://fonts.google.com/specimen/Fira+Mono

- Bootstrap

O site https://getbootstrap.com/docs/5.3/getting-started/introduction/ possui componentes que podem ser adicionados ao site, facilita a adicionar novas funções ao site.
