# 🌐 Blog Pessoal - Atividade Acadêmica

![Status](https://img.shields.io/badge/Status-Ongoing-orange)
![Versão](https://img.shields.io/badge/Versão-1.1-blue)

Projeto desenvolvido para a disciplina de **Desenvolvimento Web I** (TADS 2026). O objetivo principal foi a criação de um blog pessoal utilizando a metodologia de **Programação em Pares com Inteligência Artificial**, explorando as capacidades do Gemini AI na estruturação e estilização de páginas web.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização moderna, layouts flexíveis (Flexbox) e efeitos visuais.
- **JavaScript**: Lógica para alternância de temas (Modo Invertido).
- **Gemini AI**: Parceiro de codificação para refinamento de layout e solução de bugs.

## ✨ Funcionalidades Principais

- **Layout Responsivo**: Adaptável a diferentes tamanhos de tela.
- **Modo Dark/Light**: Botão interativo para inversão de cores.
- **Seções Dinâmicas**: Accordions para hobbies e layout hexagonal para viagens.
- **Timeline Musical**: Exibição de vídeos integrada com scroll horizontal.
- **Fundo Animado**: Background customizado com GIF para maior imersão visual.
