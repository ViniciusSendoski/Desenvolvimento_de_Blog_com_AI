# Desenvolvimento de Blog com AI

Este é um trabalho academico para a materia de Web I do ano de 2026.

Tendo como objetivo criar um blog pessoal em HTML e CSS fazendo programação em pares com uma AI.

Para o trabalho foi usado o Gemini AI 3 no modo Rápido.

Para acompanhar o a criação do blog o arquivo Documentacao.md possui os comando dados para a AI e sua saida.
